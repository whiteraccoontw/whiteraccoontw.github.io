Commission Details

Reference Materials

- Please provide a front view reference image of the character (T-pose, A-pose). Providing a three-view reference is even better.

- For complex symbols or patterns, please provide clear image files separately.

- If the character reference image does not meet the requirements, the estimated price may increase, and the number of communication rounds may increase.

- For furry model production, hair rendering will be simplified appropriately. Realistic fur texture cannot be achieved at this time.

- Currently, primarily accepting male models.

- Items that cannot be accepted:

　　Large-area armor, metal models

　　Realistic, photorealistic style painting

　　Non-original characters

- Commissioners should confirm whether the modeler's work style meets their preferences before deciding to commission.

Similarity to Reference Materials

- Visual style and silhouette will be restored as much as possible to the reference image, but complete replication is not possible and includes necessary adjustments for 3D rendering.

- Shadow rendering is mainly to enhance the overall visual feel and will not completely replicate the reference image content.

- Character models may have slight discrepancies due to camera perspective. Please raise questions or concerns during the plane model confirmation stage.

- Model coloring and colors are affected by rendering methods, ambient light settings, etc. The reference image will be the main guide during coloring.

Model Specifications

- 3D Model: The output file is a vrm file suitable for Warudo streaming.

　　Tested only for Warudo. Although vrm files are compatible with VR Chat,
　　their usability on the VRC platform is not guaranteed, and related technical support is not provided.

- Basic bone rigging, application of swaying physics.

- Basic mouth shapes (A, E, I, O, U).

- Basic expressions (joy, anger, sorrow, happiness).

- Please refer to the price list for advanced items.

Production Process

Modeling and Texturing Process All Done in Blender

1. Plane Model Stage: Fully custom modeling, referencing the outline and muscle/bone texture of the reference image; this is a crucial checkpoint.

2. UV Mapping Stage: UV unwrapping, organizing texture part distribution, reference image color analysis.

3. Rigging and Weight Painting Stage: Applying vrm rig, setting up swaying bones, weight painting.

4. Shape Key Creation Stage: Adjusting facial expressions, mouth shapes, and other deformations.

5. Before Blender Export: File organization, baking materials, preliminary testing.

6. Unity Stage: Importing textures and setting up toon shader, shape key setup, skeleton setup, physics-based swaying setup, iterative testing.

7. Warudo Testing Stage: Facial capture testing, expressions, actual swaying effect testing.

Modifications and Communication

- Each stage involves changes that are difficult to revert to varying degrees, so please pay close attention during checkpoints.

- Production time depends on the model's complexity, estimated at 6 to 10 weeks, with weekly progress updates.

- Minor revisions are accepted within 14 days after model delivery. After the deadline or once revisions are complete, the commission will be officially closed.

- The completed model will be included in the modeler's portfolio.
